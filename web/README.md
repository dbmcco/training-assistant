# Training Assistant Web

React + TypeScript + Vite frontend for the Training Assistant dashboard, plan
calendar, races, profile, and chat.

## Run

```bash
npm install
npm run dev -- --host 0.0.0.0 --port 4100
```

The Vite dev server proxies `/api` requests to `http://127.0.0.1:8000`.

## Build

```bash
npm run build
```
